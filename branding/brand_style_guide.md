
# Accutek Solar Brand Style Guide

This guide outlines the visual identity for Accutek Solar, ensuring a consistent and professional brand presence across all communications.

## 3. Typography

Consistent typography helps maintain a professional and unified brand voice.

### Primary Typeface: Montserrat

Montserrat is a modern, clean, and highly legible sans-serif typeface. It should be used for all headings, subheadings, and body text.

- **Headings:** Montserrat Bold
- **Subheadings:** Montserrat SemiBold
- **Body Text:** Montserrat Regular

### Fallback Typeface

If Montserrat is unavailable, use Arial as a substitute.
